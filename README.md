# Website Cloning & SMB Vulnerability Scanning Lab

This repository documents two practical cybersecurity labs:
1. Website Cloning
2. SMB Vulnerability Scanning using Enum4Linux

The labs were completed as part of a hands-on ethical hacking assignment
under the guidance of ParoCyber.

---

## Objectives
- Understand how website cloning works at a technical level
- Practice ethical usage of website cloning tools
- Enumerate SMB services using Enum4Linux
- Identify common SMB misconfigurations
- Document findings clearly and professionally

---

## Ethical Disclaimer
All activities documented in this repository were performed in a controlled
lab environment for **educational and authorized testing purposes only**.
No unauthorized systems were targeted.

---

## Labs Included
- Website Cloning Lab
- SMB Vulnerability Scanning (Enum4Linux)

---

## Tools Used
- Kali Linux
- Website cloning tools (demonstrated in class)
- Enum4Linux
- Metasploitable / SMB-enabled test target
